# Expense Tracker (Vanilla JS)

Simple Expense Tracker using LocalStorage.

How to run:
1. Open `index.html` in the folder `bhs indo/expense-tracker` in your browser.
2. Add transactions using the form. Data is saved automatically in LocalStorage.

Features:
- Add title, amount, category, and date
- View total for today
- Transaction list with delete
- 7-day chart using Chart.js

Notes:
- No backend required; data is local to your browser.
